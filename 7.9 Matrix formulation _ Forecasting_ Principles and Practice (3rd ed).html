7.9 Matrix formulation

Warning: this is a more advanced, optional section and assumes knowledge of matrix algebra.

Recall that multiple regression model can be written as yt=β0+β1x1,t+β2x2,t+⋯+βkxk,t+εtyt=β0+β1x1,t+β2x2,t+⋯+βkxk,t+εt where εtεt has mean zero and variance σ2σ2. This expresses the relationship between a single value of the forecast variable and the predictors.

It can be convenient to write this in matrix form where all the values of the forecast variable are given in a single equation. Then y=Xβ+εy=Xβ+ε where εε has mean 00 and variance σ2Iσ2I. Note that the XX matrix has TT rows reflecting the number of observations and k+1k+1 columns reflecting the intercept which is represented by the column of ones plus the number of predictors.

Least squares estimation

Least squares estimation is performed by minimising the expression ε′ε=(y−Xβ)′(y−Xβ)ε′ε=(y−Xβ)′(y−Xβ). It can be shown that this is minimised when ββ takes the value ^β=(X′X)−1X′y.β^=(X′X)−1X′y. This is sometimes known as the “normal equation”. The estimated coefficients require the inversion of the matrix X′XX′X. If XX is not of full column rank then matrix X′XX′X is singular and the model cannot be estimated. This will occur, for example, if you fall for the “dummy variable trap”, i.e., having the same number of dummy variables as there are categories of a categorical predictor, as discussed in Section 7.4.

The residual variance is estimated using ^σ2e=1T−k−1(y−X^β)′(y−X^β).σ^e2=1T−k−1(y−Xβ^)′(y−Xβ^).

Fitted values and cross-validation

The normal equation shows that the fitted values can be calculated using ^y=X^β=X(X′X)−1X′y=Hy,y^=Xβ^=X(X′X)−1X′y=Hy, where H=X(X′X)−1X′H=X(X′X)−1X′ is known as the “hat-matrix” because it is used to compute ^yy^ (“y-hat”).

If the diagonal values of HH are denoted by h1,…,hTh1,…,hT, then the cross-validation statistic can be computed using CV=1TT∑t=1[et/(1−ht)]2,CV=1T∑t=1T[et/(1−ht)]2, where etet is the residual obtained from fitting the model to all TT observations. Thus, it is not necessary to actually fit TT separate models when computing the CV statistic.

Forecasts and prediction intervals

Let x∗x∗ be a row vector containing the values of the predictors (in the same format as XX) for which we want to generate a forecast. Then the forecast is given by ^y=x∗^β=x∗(X′X)−1X′yy^=x∗β^=x∗(X′X)−1X′y and the estimated forecast variance is given by ^σ2e[1+x∗(X′X)−1(x∗)′].σ^e2[1+x∗(X′X)−1(x∗)′]. A 95% prediction interval can be calculated (assuming normally distributed errors) as ^y±1.96^σe√1+x∗(X′X)−1(x∗)′.y^±1.96σ^e1+x∗(X′X)−1(x∗)′. This takes into account the uncertainty due to the error term εε and the uncertainty in the coefficient estimates. However, it ignores any errors in x∗x∗. Thus, if the future values of the predictors are uncertain, then the prediction interval calculated using this expression will be too narrow.
